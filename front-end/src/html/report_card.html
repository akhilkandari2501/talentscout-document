About the Report

This report provides a detailed analysis of the candidate's performance on different aspects of thier responces.The evaluation report encompass three main areas for each question, i.e. 1. Completeness 2. Correctness 3.Communication.

This report has the following sections:

  • The Insights section provides an overall snapshot of the candidate's performance in three different feild. It includes a graphical representation of the test scores and the subsection scores.
  • The Detailed Response section provides detaied feedback and score of each question in different aspects .

Score Interpretation

Each Mock Interview test scores are on a scale of 0-100. There are 3 questions each of 30 marks .10 marks for overall communication. Out of 30 marks in each question, 15 marks for communication and 15 marks for correctness.
